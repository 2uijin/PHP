  쿠키
  setcookie()함수
  bool setcookie (string name , string value, int expire, string path )
  name : 쿠키 이르밍며 대문자와 소문자를 구분
  value : 쿠키값이며 사요자의 컴퓨터에 저장된다.
  expire : 쿠키유효 시간이며 초 단위로 표시함 (파일의 유효시간을 정함) ex:time()+60*60*24*30 >> 30동안 쿠키값 유지 )
